# Brand assets

Drop production fonts here when you license them:

- `fonts/EditorsNote-Regular.woff2`, `EditorsNote-Italic.woff2`, `EditorsNote-Semibold.woff2`
- `fonts/SequelSansDisplay-Regular.woff2`, `SequelSansDisplay-Medium.woff2`, `SequelSansDisplay-Semibold.woff2`

Then uncomment the `localFont` block in `app/layout.tsx`. Until those files exist,
the app falls back to Instrument Serif + Geist Sans (close-enough free Google fonts).
